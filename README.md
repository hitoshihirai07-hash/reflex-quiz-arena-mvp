# Reflex Quiz Arena（MVP）

「問題に早く正解した側が攻撃権」＋「反射（SPACE）で被ダメ軽減」の対戦プロトタイプです。
ポケモン/ドラクエとは無関係の新規コンセプトとして作ってあります。

## 含まれるもの
- `index.html` / `style.css` / `app.js`：ブラウザで動くMVP
- `data/questions.json`：問題100問（計算40 / 記憶30 / 論理30）
  - 計算は **％なし**
  - 2桁×2桁は **11〜19の平方のみ**
- `data/bosses.json`：ストーリーボス3体（平方/記憶/論理）

## 起動方法
ローカルで `index.html` を直接ダブルクリックすると、ブラウザによっては `fetch` がブロックされます。
以下のどちらかで起動してください。

### 1) Pythonで簡易サーバー（例）
プロジェクトフォルダで：
- `python -m http.server 8000`

その後ブラウザで：
- `http://localhost:8000`

### 2) GitHub Pages / Cloudflare Pages に配置
このフォルダ内容をそのまま配置すれば動きます（`/data/*.json` を同一階層に置く）。

## 操作
- 4択：クリック / キー(1〜4)
- 数字入力：入力して Enter
- 反射：**SPACE**（タイミングで成功/ジャスト）

## ルール（MVP）
- HP：P1=100、CPU/ボスは100〜140
- 両者不正解：ノーダメ
- ダメージ：問題難易度(1..4)に応じて 8/10/12/14 を基礎に、回答速度差で最大+40%
- 反射：
  - 失敗：軽減なし
  - 成功：被ダメ50%
  - ジャスト：被ダメ20% + 返しダメ少量
