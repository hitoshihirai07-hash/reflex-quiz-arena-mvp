ルール

10問で決着する4人スコアバトル。正解＋スピードで得点が決まります。

遊ぶ FAQ
出題
練習では「計算だけ」「記憶だけ」「論理だけ」のジャンル別モードも選べます。
  • 1試合は10問
  • ジャンル：計算 / 記憶 / 論理
  • 形式：4択 と 数字入力
  • 問題は固定データに加えて、条件内で自動生成される場合があります（％なし / 2桁×2桁なし）
  • 計算は％なし、2桁×2桁は11〜19の平方のみ
スコア
  • 正解：+100
  • 早さボーナス：最大 +40（早いほど加点）
  • 不正解：0

同点は「合計スコア → 正解数 → 総回答時間（短い方）」で判定します。

フリー / レートの違い
  • フリー対戦：ランダム。対戦後のレート更新なし。
  • レート対戦：近い実力同士。順位に応じてポイント獲得（減点なし）。格下勝ちは追加ボーナス。
広告スロット（説明ページ側）
ゲーム操作の近くには置かない前提です。
